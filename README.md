## What I learned

- In this project, I used html{font-size 62.5%} to calculate px for rem more easily

```css
html {
  /* just to make it easier to calculate px for rem 
    
  calculation: 10px / 16px = 0,625 / 100
    
  10px = 1rem
  */
  font-size: 62.5%;
}
```
